# ArduinoUnoR4WiFiWeatherStation
An Arduino Uno R4 WiFi sketch to download weather data and display it on the LED Matrix.

1. Open this project in your Arudino IDE.
2. Download the required libraries. Using their README files to install properly.
   1. https://github.com/DTSavaria/ArduinoLedMatrixAsyncScrollingMessage
   2. https://github.com/DTSavaria/ArduinoWeatherDataDownloader
3. Check the CUSTOMIZE OPTIONS section in the ino file. Most importantly setting the WiFi and location details.
4. Optional: Connect a Thermo Modulino to read indoor temperature.
4. Upload it to your Arduino.
